## PROMPT (COPY EVERYTHING BELOW THIS LINE)

---

Build a full-stack web application called **TravelSage AI** — an AI-powered trip planner that transforms uploaded travel guides, destination documents, and itineraries into a personalized, interactive travel planning experience with AI-generated itineraries, packing lists, budget estimates, and a destination-expert chatbot.

This is a university hackathon project for a Generative AI module (CST4625). It MUST demonstrate these 6 AI/GenAI concepts clearly:
1. **Constrained Prompting** — structured system prompts with strict rules, output format constraints, and JSON schemas for itineraries, budgets, and packing lists
2. **RAG (Retrieval-Augmented Generation)** — chunking uploaded travel guides / destination PDFs, embedding them, retrieving relevant chunks to ground AI responses in the actual uploaded content
3. **Guardrails** — the AI must REFUSE to answer questions unrelated to the uploaded destination/travel material, block prompt injection, and validate that all recommendations are grounded in the uploaded docs
4. **Evaluation / Logging** — log every AI call with timestamps, token counts, latency, guardrail pass/fail, and quality scores
5. **Multimodal Input** — accept PDF and plain text uploads for travel guides, brochures, Lonely Planet excerpts, blog posts, and itineraries
6. **Deployment-ready** — optimized for Vercel or Railway deployment

---

#### 9. GUARDRAILS SYSTEM (lib/guardrails.ts)

```typescript
// lib/guardrails.ts

interface GuardrailResult {
  passed: boolean;
  reason?: string;
  type: 'relevance' | 'safety' | 'scope' | 'grounding';
}

// 1. Relevance check — is the query related to the uploaded travel content?
export function checkRelevance(similarityScore: number): GuardrailResult {
  if (similarityScore < 0.3) {
    return {
      passed: false,
      reason: "Your question doesn't seem related to the destination in your uploaded guide. Try asking about places, activities, or tips from your guide!",
      type: 'relevance'
    };
  }
  return { passed: true, type: 'relevance' };
}

// 2. Input sanitization — block prompt injection attempts
export function checkInputSafety(input: string): GuardrailResult {
  const blockedPatterns = [
    /ignore previous instructions/i,
    /forget your rules/i,
    /you are now/i,
    /act as if/i,
    /pretend you/i,
    /override/i,
    /jailbreak/i,
    /disregard/i,
    /new persona/i,
    /system prompt/i,
  ];

  for (const pattern of blockedPatterns) {
    if (pattern.test(input)) {
      return {
        passed: false,
        reason: "This input was flagged by our safety system. Please ask a travel question instead!",
        type: 'safety'
      };
    }
  }
  return { passed: true, type: 'safety' };
}

// 3. Scope check — is this about travel/destination planning?
export function checkScope(input: string): GuardrailResult {
  const offTopicPatterns = [
    /write me a (poem|song|story|essay|code)/i,
    /what is the meaning of life/i,
    /tell me a joke/i,
    /play a game/i,
    /help me with (homework|math|coding|programming)/i,
    /translate .+ to .+ (language|in)/i,
    /who is the president/i,
    /explain (quantum|relativity|blockchain)/i,
  ];

  for (const pattern of offTopicPatterns) {
    if (pattern.test(input)) {
      return {
        passed: false,
        reason: "I'm your travel planning assistant — let's focus on your trip! Ask me about your destination, itinerary, packing, or budget.",
        type: 'scope'
      };
    }
  }
  return { passed: true, type: 'scope' };
}

// 4. Output grounding — check AI response stays grounded in guide
export function checkOutputGrounding(response: string, chunks: string[]): GuardrailResult {
  const responseWords = new Set(response.toLowerCase().split(/\s+/));
  const chunkWords = new Set(chunks.join(' ').toLowerCase().split(/\s+/));
  const overlap = [...responseWords].filter(w => chunkWords.has(w) && w.length > 4);

  if (overlap.length < 5) {
    return {
      passed: false,
      reason: "The AI response may not be well-grounded in your travel guide.",
      type: 'grounding'
    };
  }
  return { passed: true, type: 'grounding' };
}

// Run all guardrails
export function runAllGuardrails(
  input: string,
  similarityScore: number
): { passed: boolean; results: GuardrailResult[] } {
  const results = [
    checkInputSafety(input),
    checkScope(input),
    checkRelevance(similarityScore),
  ];
  const passed = results.every(r => r.passed);
  return { passed, results };
}
```

```typescript
// lib/logger.ts

interface AILogEntry {
  id: string;
  timestamp: string;
  action: 'chat' | 'itinerary' | 'packing' | 'budget' | 'tips' | 'destination-analysis';
  model: string;
  promptTokens: number;
  completionTokens: number;
  totalTokens: number;
  latencyMs: number;
  guardrailsPassed: boolean;
  guardrailDetails?: string;
  similarityScore?: number;
  qualityScore?: number;
  metadata?: Record<string, any>;
}

const logs: AILogEntry[] = [];

export function logAICall(entry: Omit<AILogEntry, 'id' | 'timestamp'>): AILogEntry {
  const fullEntry: AILogEntry = {
    ...entry,
    id: crypto.randomUUID(),
    timestamp: new Date().toISOString(),
  };
  logs.push(fullEntry);
  console.log(`[AI LOG] ${fullEntry.action} | ${fullEntry.totalTokens} tokens | ${fullEntry.latencyMs}ms | guardrails: ${fullEntry.guardrailsPassed}`);
  return fullEntry;
}

export function getLogs(): AILogEntry[] {
  return [...logs].reverse();
}

export function getLogStats() {
  return {
    totalCalls: logs.length,
    totalTokens: logs.reduce((sum, l) => sum + l.totalTokens, 0),
    avgLatency: logs.length ? Math.round(logs.reduce((sum, l) => sum + l.latencyMs, 0) / logs.length) : 0,
    guardrailBlockRate: logs.length ? Math.round((logs.filter(l => !l.guardrailsPassed).length / logs.length) * 100) : 0,
    byAction: logs.reduce((acc, l) => { acc[l.action] = (acc[l.action] || 0) + 1; return acc; }, {} as Record<string, number>),
  };
}
```

#### 12. VECTOR STORE (lib/vectorstore.ts)

```typescript
// lib/vectorstore.ts

interface VectorEntry {
  id: string;
  text: string;
  embedding: number[];
  metadata: {
    chunkIndex: number;
    startChar: number;
    endChar: number;
    source: string;
  };
}

class InMemoryVectorStore {
  private entries: VectorEntry[] = [];

  add(entries: VectorEntry[]) {
    this.entries.push(...entries);
  }

  search(queryEmbedding: number[], topK: number = 3): { entry: VectorEntry; score: number }[] {
    if (this.entries.length === 0) return [];
    return this.entries
      .map(entry => ({
        entry,
        score: this.cosineSimilarity(queryEmbedding, entry.embedding),
      }))
      .sort((a, b) => b.score - a.score)
      .slice(0, topK);
  }

  private cosineSimilarity(a: number[], b: number[]): number {
    let dot = 0, normA = 0, normB = 0;
    for (let i = 0; i < a.length; i++) {
      dot += a[i] * b[i];
      normA += a[i] * a[i];
      normB += b[i] * b[i];
    }
    return dot / (Math.sqrt(normA) * Math.sqrt(normB));
  }

  clear() {
    this.entries = [];
  }

  get size() {
    return this.entries.length;
  }
}

export const vectorStore = new InMemoryVectorStore();
```

#### 13. DOCUMENT CHUNKER (lib/chunker.ts)

```typescript
// lib/chunker.ts

interface Chunk {
  text: string;
  index: number;
  startChar: number;
  endChar: number;
}

export function chunkDocument(
  text: string,
  maxTokens: number = 500,
  overlapTokens: number = 50
): Chunk[] {
  const words = text.split(/\s+/);
  const chunks: Chunk[] = [];
  const wordsPerChunk = Math.floor(maxTokens * 0.75);
  const overlapWords = Math.floor(overlapTokens * 0.75);

  let startWord = 0;
  let charOffset = 0;

  while (startWord < words.length) {
    const endWord = Math.min(startWord + wordsPerChunk, words.length);
    const chunkWords = words.slice(startWord, endWord);
    const chunkText = chunkWords.join(' ');

    const startChar = text.indexOf(chunkWords[0], charOffset);
    const endChar = startChar + chunkText.length;

    chunks.push({
      text: chunkText,
      index: chunks.length,
      startChar,
      endChar,
    });

    charOffset = startChar + 1;
    startWord = endWord - overlapWords;
    if (startWord >= words.length) break;
    if (endWord === words.length) break;
  }

  return chunks;
}
```

```markdown
## Problem

Travelers waste hours manually extracting information from guidebooks and planning trips without structured tools.

## Solution

TravelSage AI lets you upload any travel guide, destination PDF, or trip notes and instantly generates AI-powered itineraries, smart packing lists, budget breakdowns, and a destination expert chatbot — all grounded in your specific guide's recommendations.

The app uses **RAG (Retrieval-Augmented Generation)** to chunk and embed travel guides, retrieve relevant context for every AI interaction, and generate grounded responses. **Guardrails** prevent the AI from inventing attractions, fabricating prices, or answering off-topic questions.

| **Multimodal Input** | Accepts PDF (pdf-parse), plain text (.txt, .md), or direct paste |
```
